# SpendLens - Expense Dashboard 💰

SpendLens is a simple and clean expense tracking website.  
Track your daily, weekly, and monthly expenses easily from your browser.

## Features
- **Add Expense**: Add expenses with Description, Amount, Category, and Date
- **Category Tracking**: Organize spending by Food, Transport, Bills, Business, etc.
- **Monthly Summary**: View total spent per month
- **Local Storage**: Your data is saved directly in your browser
- **Mobile Friendly**: Works on phone, tablet, and desktop

## How to Use
1.  Enter a description: e.g. `Grocery Shopping`
2.  Enter the amount in PKR
3.  Select a category
4.  Pick the date
5.  Click `Add Expense`

## Tech Stack
HTML, CSS, JavaScript
